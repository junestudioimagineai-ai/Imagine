A modern, feature-rich Social Media Marketing (SMM) marketplace with enterprise-grade features.

## Features

### User Features
- **Browse Services**: View 500+ social media services across all major platforms
- **Bundle Packages**: Save up to 50% when bundling 3+ services
- **AI Assistant**: 24/7 expert help for social media, content design, marketing, and Nigerian market insights
